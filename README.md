# Hardware-Hunter

**Hardware-Hunter** is a C++ project that demonstrates Object-Oriented Programming (OOP) concepts and file handling.  

This application allows users to:  
- Find PC parts  
- Sell or build a custom PC  
- Calculate the total price of selected components  
- Check if parts are available  

> **Purpose:** This project was created primarily as a learning exercise to practice C++, OOP, and working with text files.

---

## Features
- Uses classes and objects for clean code structure  
- Saves and loads data using text files  
- Calculates total prices dynamically  
- Basic inventory management for PC parts  

---

## Technologies
- **Language:** C++  
- **File Storage:** Text files (.txt)  
- **Concepts Used:** Object-Oriented Programming (OOP), File I/O
